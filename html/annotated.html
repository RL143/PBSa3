Data Structures
Here are the data structures with brief descriptions:
 CAngleStruct to store indices of particles in an angle i-j-k
 CBondStruct to store indices of bonded particles i-j
 CCelllistStruct used to store a cell-linked-list
 CDeltaRStruct to store a 3D vector and its square length. This is expecially useful for connecting vectors in e.g. neighbor lists
 CDihedralStruct to store indices of particles in a dihedral i-j-k-l
 CIndex3DStruct to store i, j, k indices of a 3D grid
 CNbrlistStruct to store a neighbor list
 CPairStruct to store a pair of particles: its indices and connecting vector
 CParametersStruct to store all parameters. These parameters are set by the function set_parameters
 CVec3DStruct to store x, y, and z component of a 3D vector
 CVectorsStruct with pointers to all particle arrays relevant for a MD simulation
